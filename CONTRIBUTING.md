# Contributing

Thanks for improving **awesome-gpt-image2**.

## What belongs here

- Repositories directly focused on **GPT-Image-2 / GPT Image 2 / ChatGPT Images 2.0**
- Projects where GPT-Image-2 is a meaningful product feature, workflow target, or integration surface
- Corrections to license metadata, descriptions, dead links, or categories

## What usually does not belong here

- Generic AI lists that only mention GPT-Image-2 once
- News dumps, trend mirrors, or marketing-only pages with no meaningful repository content
- Closed-source products without a public GitHub repository

## Suggested pull request format

Please include:

1. Repository URL
2. Why it belongs in this list
3. Evidence that GPT-Image-2 is central or meaningfully implemented
4. Preferred category
